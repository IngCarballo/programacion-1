📝 Trabajo Práctico N°6: Creación e Inicio de Proyecto React
🎯 Objetivo
Poner en marcha un servidor React que se ejecute en el puerto 3000 y preparar un bosquejo del diseño de la aplicación.


📦 Entregables
Proyecto React funcionando localmente en http://localhost:3000
Repositorio con commit inicial y ramas organizadas
Bosquejo o diagrama de diseño inicial


🛠️ Actividades
1. Crear y preparar el repositorio
Crear un nuevo repositorio en GitHub o tu plataforma preferida.
Clonarlo localmente.
Realizar un commit inicial en la rama main.
Crear una nueva rama para el desarrollo, por ejemplo:
feature/start-react-project.


2. Instalar Node.js
Descargar e instalar Node.js desde el sitio oficial: https://nodejs.org/
Verificar la instalación con los comandos:

node -v

npm -v


3. Crear el proyecto React
Desde la terminal, en la ubicación donde quieras el proyecto, ejecutar:

npx create-react-app nombredelproyecto

Reemplaza nombredelproyecto por el nombre que desees para tu app (en minusculas). 


4. Configurar el entorno de desarrollo
Abrir Visual Studio Code.
Instalar la extensión: ES7+ React/Redux/React-Native Snippets.
Añadir el proyecto al workspace.


5. Documentar el diseño inicial
Realizar un diagrama de la Home o estructura general del proyecto (puede ser una idea básica de componentes o flujo).
Guardar el diagrama en un directorio llamado docs/ o incluirlo en el archivo README.md.


6. Plasmar el diseño en la aplicación
Utilizando la documentación de react Quick Start – React crear la home 
Se sugiere el uso de bootstrap


✅ Resultado Esperado
Un servidor React en funcionamiento (puerto 3000).
Proyecto organizado en Git con ramas separadas.
Un bosquejo inicial del diseño guardado y versionado.
