# 📝 Trabajo Práctico N°8: Creación de Vistas con Bootstrap

## 🎯 Objetivo

Desarrollar las **vistas estáticas del proyecto frontend** utilizando el framework **Bootstrap**, siguiendo las maquetas previamente definidas y asegurando un diseño **responsive** adaptable a diferentes dispositivos.

---

## 📦 Entregables

- Vistas estáticas creadas con Bootstrap  
- Diseño responsive adaptado a móviles, tablets y PC  
- Repositorio con ramas organizadas y commits claros

---

## 🛠️ Actividades

### 1\. Crear y preparar la rama de trabajo

- Crear una nueva rama desde `main` o la base actual del proyecto, por ejemplo:  
  `feature/bootstrap-views`

---

### 2\. Integrar Bootstrap al proyecto

- Instalar Bootstrap mediante npm o incluir el CDN en el `index.html`  
- Verificar que los estilos de Bootstrap se apliquen correctamente

---

### 3\. Crear vistas estáticas

- Implementar las vistas principales del proyecto (Home, Login, Registro, etc.)  
- Basarse en las maquetas aprobadas  
- Utilizar componentes de Bootstrap:  
  - Formularios  
  - Grillas (grid system)  
  - Botones  
  - Navbars  
  - Cards u otros según necesidad

---

### 4\. Garantizar diseño responsive

- Asegurar que todas las vistas se vean correctamente en distintos tamaños de pantalla  
- Usar clases utilitarias de Bootstrap para mejorar adaptabilidad  
- Probar en dispositivos móviles o herramientas de inspección del navegador

---

### 5\. Organizar y documentar

- Agrupar las vistas dentro de una carpeta `/views` o `/pages`  
- Comentar el código y mantener una estructura clara  
- Hacer commits frecuentes con mensajes descriptivos

---

## ✅ Resultado Esperado

- Todas las vistas principales del proyecto creadas con Bootstrap  
- Interfaz visual coherente y adaptativa  
- Código limpio, organizado y versionado en el repositorio
